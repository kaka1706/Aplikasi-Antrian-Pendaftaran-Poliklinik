# Polyclinic Registration Queue Application

The Polyclinic Registration Queue Application is an information system consisting of two parts:

1. **Web Admin Panel**  
   Used by polyclinic staff/admin to:
   - View the queue list
   - Manage schedules
   - Calling & completing queues
   - Monitoring patient and service statistics

2. **Mobile App (Flutter)**  
   To be used by **users/students/patients** to:
   - Register for queues
   - View queue status
   - Receive active queue notifications

> 🔧 Currently, this repository **only contains the Web Admin section**.  
> The Flutter mobile section will be added to this repository soon.

---

## Key Features (Admin)

📌 Admin Login  
📌 View queue list per service  
📌 Filtering by schedule and clinic  
📌 Call, complete, and cancel queues  
📌 Daily/weekly queue statistics  
📌 Service schedule settings
